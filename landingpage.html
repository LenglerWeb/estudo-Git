Comandos:

git init - Inicia uma linha do tempo.

git add . - Adiciona ou Atualiza mudanças para irem para a linha do tempo.

git commit -m "texto" - Adiciona um ponto na linha do tempo

git log - Visualiza os pontos linha do tempo / commit

git status - Informa o estado das alterações do projeto

git show - Apresenta determinado ponto na história

git branch - Gerencias novas linhas do tempo

git checkout - Manipula a linha do tempo

git merge - Unir as linhas do tempo

git push - Envia alterações locais para o repositório remoto
